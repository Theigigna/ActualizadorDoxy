Detailed Description

A set of rendering that is cached onto the graphics card and hopefully is quicker to render. Note that there are some things that can't be done in lists and that all dependent operations must be container. For instance, any colour configuration can not be assumed from outside the cache.

Note: The destroy method needs to be used to tidy up. This is pretty important in this case since there are limited number of underlying resources.
