Detailed Description

An effect that has a number of configuration values. This allows the effect to be configured in the Hiero GUI and to be saved and loaded to and from a file.

List org.newdawn.slick.font.effects.ConfigurableEffect.getValues ( )

Returns the list of Values for this effect. This list is not typically backed by the effect, so changes to the values will not take affect until setValues(List) is called.
