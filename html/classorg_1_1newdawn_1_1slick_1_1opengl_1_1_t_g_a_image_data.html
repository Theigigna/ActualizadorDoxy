Detailed Description

A utility to load TGAs. Note: NOT THREAD SAFE

Fresh cut of code but largely influeneced by the TGA loading class provided as part of the Java Monkey Engine (JME). Why not check out what they're doing over at http://www.jmonkeyengine.com. kudos to Mark Powell.

Author
Kevin Glass
