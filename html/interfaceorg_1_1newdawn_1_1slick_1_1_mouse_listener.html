Notification that a mouse button was clicked. Due to double click handling the single click may be delayed slightly. For absolute notification of single clicks use mousePressed().

To be absolute this method should only be used when considering double clicks

Parameters
buttonThe index of the button (starting at 0)
xThe x position of the mouse when the button was pressed
yThe y position of the mouse when the button was pressed
clickCountThe number of times the button was clicked
