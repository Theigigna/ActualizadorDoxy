Detailed Description

A utility to allow performing GL operations without contaminating the Slick OpenGL state. Note this will not protect you from OpenGL programming errors like a glBegin() without a glEnd(), or glPush() without glPop() etc.

Expected usage:

SlickCallable callable = new SlickCallable() { public performGLOperations() throws SlickException { GL.glTranslate(0,0,1); glBegin(GL.GL_POLYGONS); glVertex(..); ... glEnd(); } } callable.call();

Alternatively you can use the static methods directly

SlickCallable.enterSafeBlock();

GL.glTranslate(0,0,1); glBegin(GL.GL_POLYGONS); glVertex(..); ... glEnd();

SlickCallable.leaveSafeBlock();
